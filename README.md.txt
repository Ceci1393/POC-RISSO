# PoC - Generación de Respuestas Automáticas para FAQs de Tarjetas de Crédito

Este proyecto utiliza la API de OpenAI para generar respuestas automáticas a preguntas frecuentes sobre tarjetas de crédito. El script está desarrollado en un Jupyter Notebook y exporta los resultados en un archivo CSV.

## Tecnologías Usadas
- Python
- Jupyter Notebook
- OpenAI API
- Pandas
